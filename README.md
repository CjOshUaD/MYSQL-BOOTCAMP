# MySQL Bootcamp 📊✍️

Welcome to the **MySQL Bootcamp** repository! This repository contains all the code, practice exercises, queries, and projects developed throughout my journey of mastering SQL and MySQL database design. 

The goal of this bootcamp is to go from an absolute beginner in relational databases to an expert capable of designing database schemas, writing complex queries, and optimizing database performance.

---

## 🧭 Course Subjects

The repository covers a structured learning path divided into several foundational and advanced database concepts:

* **Database Basics & CRUD:** Creating, reading, updating, and deleting databases and tables.
* **Data Types:** Mastering `INT`, `VARCHAR`, `DECIMAL`, `DATE`, `TIME`, `DATETIME`, and `TIMESTAMP`.
* **String Functions:** Constructing queries using `CONCAT`, `REPLACE`, `SUBSTRING`, `REVERSE`, and `CHAR_LENGTH`.
* **Refining Selections:** Using `DISTINCT`, `ORDER BY`, `LIMIT`, and pattern matching with `LIKE`.
* **Aggregate Functions:** Analyzing data using `COUNT`, `SUM`, `MIN`, `MAX`, `AVG`, and grouping records with `GROUP BY` and `HAVING`.
* **Comparison & Logical Operators:** Building logical filtering rules using `AND`, `OR`, `NOT`, `BETWEEN`, `IN`, and `CASE` statements.
* **Constraints & Altering Tables:** Enforcing data integrity with `PRIMARY KEY`, `FOREIGN KEY`, `UNIQUE`, `CHECK`, and updating schemas via `ALTER TABLE`.
* **Relationships & JOINs:** Connecting multiple data sources using `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, and `CROSS JOIN`.

---

## 🚀 Project Highlights

Inside this bootcamp, you will find practical applications of database modeling, including:
1. **Instagram Clone Database:** A comprehensive, multi-table database schema mirroring Instagram's core structure—tracking users, photos, comments, likes, follows, and hashtags.
2. **Node.js & MySQL Integration:** Connecting a local MySQL database with an Express.js backend, using packages like `faker.js` to generate mass fake data for scalability testing.
